Welcome to my portfolio !

I'm a french ph.D student in computer vision and deep learning at the University of technology of Troyes. I'm currently working in the domain of industrial unsupervised anomaly detection especially for textures.

My projects 🛠

Industrial Defect Lib

This project introduce a new library dessigned to generate synthetic defects based on proposed methods from the litterature. Currently, the library is able to generate defects for the following defects : NSA, textural, structural, blur and cutpaste.


💻 Github Page
🤗 HuggingFace Demo

Industrial textile dataset

This project introduce a new dataset dessigned for fabric anomaly detection. It contains 10 categories of textile with a wide variety of defects extracted from a textile factory with high resolution cameras.
